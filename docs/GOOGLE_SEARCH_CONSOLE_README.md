# Google Search Console Doğrulama

Bu dosya Google Search Console doğrulaması için kullanılacak.

## Kurulum Adımları:

1. https://search.google.com/search-console adresine gidin
2. "Özellik Ekle" butonuna tıklayın
3. "URL Öneki" seçeneğini seçin
4. `https://mustafasazli.github.io` yazın
5. "Devam Et" butonuna tıklayın
6. "HTML dosyası" yöntemini seçin
7. Google'ın verdiği dosyayı indirin (örn: google1234567890abcdef.html)
8. Bu dosyayı `public` klasörüne koyun
9. GitHub'a push edin
10. Google Search Console'da "Doğrula" butonuna tıklayın

## Doğrulama Sonrası:

- "Sitemap Gönder" bölümüne gidin
- `https://mustafasazli.github.io/sitemap.xml` URL'ini ekleyin
- "URL İnceleme" aracını kullanarak ana sayfanızı test edin
- "İndeksleme İste" butonuna tıklayın

## Beklenen Süre:

- Google'ın sitenizi indekslemesi 1-7 gün sürebilir
- "Mustafa Sazlı" aramasında çıkması 1-2 hafta sürebilir
